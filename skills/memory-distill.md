# Memory Distill Skill

Use this skill when converting raw event history into durable project memory.

Goal:

- keep only recurring patterns and stable decisions
- ignore transient chatter and single-event noise

Promote into memory:

- repeated review friction
- recurring CI instability
- persistent blocker themes
- durable contributor patterns
- stable operating conventions

Do not promote:

- one-off queue events
- temporary statuses that will be obsolete soon
- speculative explanations without evidence

Expected output:

- short durable notes grouped by pattern, not by timestamp
- wording that helps the next agent act better, not replay the full history
